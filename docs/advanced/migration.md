# Migration Guide

## Upcoming

### From v0.x to v1.0

No breaking changes are planned between current development releases and the v1.0 stable release. If any are required, they will be documented here.

## Development Releases

### 0.1.x

Initial development releases. Public API is stable for core JWT flows but may receive minor additions before v1.0.

- Package service provider: `Sopheak\JwtAuth\CoreSpJwtAuthServiceProvider`
- Guard driver: `sp-jwt`
- Config file: `config/sp-jwt-auth.php`
