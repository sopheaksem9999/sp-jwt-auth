# Code of Conduct

## Our Pledge

We want this project to be a practical, respectful place for Laravel developers to collaborate on authentication infrastructure.

Project participants are expected to:

- Be respectful and constructive.
- Keep feedback focused on the code, docs, behavior, or security impact.
- Assume good intent while still asking for clear technical evidence.
- Avoid harassment, insults, discrimination, or personal attacks.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, threats, or abusive language.
- Discriminatory comments or imagery.
- Publishing private information without explicit permission.
- Repeated off-topic disruption.
- Sharing real secrets, customer data, or sensitive security details in public threads.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or discussions that violate this code of conduct. Maintainers may also temporarily or permanently block contributors who repeatedly violate it.

Report conduct concerns privately to the project maintainers. Security issues should follow `SECURITY.md` instead.
